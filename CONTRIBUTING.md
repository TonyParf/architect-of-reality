# Contributing to Architect of Reality

This project is an open philosophical framework. Contributions welcome.

## Ways to Contribute

- **Translations** — The framework exists in English, Ukrainian, Russian, Croatian. New languages welcome.
- **Academic critique** — Rigorous pushback on the Grand Axiom or the 17 Foundations
- **Applications** — Case studies of the framework applied to real systems
- **Fiction** — The Anthosphere fictional universe (see anthosphere.com)

## Contact

Open an issue or reach out via [anthosphere.com](https://anthosphere.com)
